# 🥖 Panadería Rosy

> **Nota:** Este proyecto fue creado con asistencia de Inteligencia Artificial.

## Descripción

Sitio web oficial de **Panadería Rosy**, una panadería artesanal con más de 25 años de tradición en Tecomatlán, Puebla. El sitio presenta la historia, productos y servicios de contacto de manera elegante y accesible.

## 🎯 Características

- **Diseño responsivo**: Se adapta perfectamente a dispositivos móviles, tablets y computadoras de escritorio
- **Navegación intuitiva**: Menú principal con enlaces internos a cada sección
- **Sección Hero**: Presentación impactante con imagen de productos
- **Nuestra Historia**: Relato sobre la tradición y valores de la panadería
- **Catálogo de Productos**: 4 líneas de productos con detalles
- **Contacto**: Información de ubicación, teléfono, correo y formulario
- **WhatsApp flotante**: Botón para contacto directo vía WhatsApp
- **Animaciones sutiles**: Efectos reveal al hacer scroll

## 🚀 Cómo Usar

### En local
1. Descarga o clona los archivos del proyecto
2. Abre `index.html` en tu navegador
3. El sitio se mostrará completamente funcional

### En producción
1. Sube los archivos a tu servidor web
2. Asegúrate de que todas las rutas de imágenes sean correctas
3. Personaliza los datos de contacto según sea necesario

## 🎨 Personalización

### Cambiar imágenes
Reemplaza los archivos `.jpg` en la carpeta del proyecto:
- `CCC.jpg` → Imagen hero
- `Cuernitos.jpg` → Imagen de historia

### Actualizar información de contacto
En el archivo `index.html`, busca y modifica:
- Teléfono: Línea 355 (teléfono principal)
- Correo: Línea 367 (correo de contacto)
- Ubicación: Línea 342 (dirección)
- Horario: Línea 373 (horarios de atención)

### Integrar Google Maps
En la sección de contacto, reemplaza el iframe con tu embed de Google Maps:
```html
<iframe src="https://www.google.com/maps/embed?pb=TU_EMBED_ID" ...></iframe>
```
